Presesenting TAG-Cas:
A 3D-aware Deep Learning model for predicting CRISPR-Cas9 off-targets. 
Integrates 1D sequence, 200bp genomic context, and explicit 3D epigenetics (Hi-C/ATAC-seq)
via Multiplicative Biological Gating to enforce true chromatin accessibility.
